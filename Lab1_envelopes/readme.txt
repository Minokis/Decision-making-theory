Задание 2: Парадокс с 2 конвертами https://ru.wikipedia.org/wiki/Задача_о_двух_конвертах

На занятии Лобанов привел нас к выводу, что для корректной формулировки задачи не хватает хотя бы плотности
распределения вероятностей p(x), где x - сумма денег в одном из конвертов.
Задание заключается в том, чтобы написать симуляцию игры с конвертами с какой-нибудь заданной плотностью
распределения (например, с какой-нибудь экспонентой с коэффициентами и константами). Опять-таки, проанализировать и
показать, при каких x (сумма, которую вы нашли в конверте) вам лучше поменяться с соседом.
Можно нарисовать график мат.ожидания выигрыша при смене конверта от x, для наглядности.

Советуемые инструменты: Инструменты: Python + numpy + matplotlib для графиков по желанию.
Идея: попробовать метод обратных функций
